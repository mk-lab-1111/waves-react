# 凪 NAGI - App (試作版)

バイノーラルビート × メディテーション PWA の試作実装です。

## 起動方法

```bash
npm install
npm run dev
```

`http://localhost:5173` を開いてください。

## 実装済み機能

- ホーム画面（Natureシリーズ5曲 + Storyシリーズ近日公開）
- 「深海」（無料トラック）の実再生
  - **Soft / Deep トグル**：バイノーラルビートの強さを切り替え（音源を即座に差し替え、再生位置を維持）
  - リアルタイムの残り時間表示
  - 「Meditate / Pause」ボタン
- 呼吸法サポート（3-6呼吸法）：実際に動く呼吸ガイド + 5分カウントダウン
- 仕組みページ（バイノーラルビートの説明 + 周波数帯ガイド）
- ロックされたトラック・プレミアムバナーのタップ → アラート表示（プレースホルダー）

## 音源ファイル

`public/audio/` に以下を配置済み：
- `深海の静けさ_trial_soft.mp3`（バイノーラル強度 0.33）
- `深海の静けさ_trial_deep.mp3`（バイノーラル強度 0.5）

新しい曲を追加する場合は、同様に `public/audio/` へ配置し、`src/data/tracks.js` に
`audio: { soft: '...', deep: '...' }` を追加してください。
